Today, CIM and WBEM do not have any discovery tools.  You must specifically setup the URL and Port number of the WBEM Services Broker you want to query.  The defaults are provided for use in a test environment with the Pegasus Services Broker.

Pegasus URL - Enter the URL of the CIM Server you want to access here.  Note that this will be the same URL as the WEB server accessed by these forms.

Host Port -  A CIM Server must have at least one port and may have more than one.  The default port used (effective with the CIM Schema version 2.5) is 5988.  This port has been reserved by the DMTF for use of CIM Servers.

User Password -

Debug - Debug is a tool to aid testing between the CIM Forms, the program CGIClient, the CIM Server.  It simply provides a way to turn on debug traces in CGIClient.  If you want to set debug strings, simply enter the list of keywords. 

The keywords are:

  • all - Turn on all trace
  • query - Show the string input to CGI Client
